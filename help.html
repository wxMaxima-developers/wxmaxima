Getting help

To get support for using Maxima, use the mailing lists. Questions specific to wxMaxima can be posted to a forum.

Math fonts

wxMaxima can use jsMath TeX fonts to display math output. The fonts are used to display Greek characters and math symbols (like the integral and sum symbols). Follow these instructions to install jsMath fonts in your computer.
