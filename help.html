Getting help

To get support for using Maxima, use the mailing lists. Questions specific to wxMaxima can be posted to a forum. A lot of documentation can also be found at maxima webpages.

Math fonts

wxMaxima can use jsMath TeX fonts to display math output. The fonts are used to display Greek characters and math symbols (like the integral and sum symbols). Follow these instructions to install jsMath fonts in your computer.

The offline manuals

Maxima (the program that does the actual mathematics) and wxMaxima (the graphical user interface) both come with their own documentation that is accessible without the need of having access to the internet. Both manuals are accessible online, too:

  • The manual Maxima, in various formats, is here
