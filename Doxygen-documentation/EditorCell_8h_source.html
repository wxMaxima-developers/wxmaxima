76 // The selection/cursor setters clamp their argument to a valid index into the
77 // current text via ClampToText(), so m_selectionStart / m_selectionEnd are
78 // always in [0, m_text.Length()] (e.g. CursorMove(-n) past the start clamps to
79 // 0 instead of wrapping a size_t to a huge value). The getters below keep a
80 // defensive std::min as well, because the text can still shrink *after* a
81 // selection was set without the selection being updated.
